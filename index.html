jQuery Scrollable

A jQuery plugin to create iOS / Mac OS X Lion style fading scrollbars for scrollable elements.

Dependencies

To use jQuery Scrollable please make sure to include the following in your project:

  • jQuery (1.7.2+)
  • jQuery (1.8.18+) UI — This is needed for draggables.

Note: This plugin may work with older versions of jQuery but was developed using the above.

Options

align (Default: 'right')
This option determines the alignment of the scrollbar.
Accepted values: 'right', 'left'

fade (Default: true)
Set to true to fade the scrollbar in and out on hover. If set to false, the scrollbar will always be visible.
Accepted values: true, false

margin (Default: 0)
This determines the margin (in pixels) around the track of the scrollbar.
Note: This must be an integer value.

mousewheel (Default: true)
Set to true to allow scrolling with the mousewheel.
Accepted values: true, false

mousewheel_step (Default: 32)
The distance (in pixels) that will be scrolled on each step in the rotation of the mousewheel.
Note: This must be an integer value.

show_track (Default: false)
Set to true to show the track. If set to false, the track will be hidden.
Accepted values: true, false

thumb_class (Default: 'scrollable_thumb')
This is the class name that will be assigned to the thumb of the scrollbar.

track_class (Default: 'scrollable_track')
This is the class name that will be assigned to the track of the scrollbar.

track_width (Default: 8)
The width (in pixels) of the track of the scrollbar.
Note: This must be an integer value.

wrapper_class (Default: 'scrollable_wrapper')
This is the class name that will be assigned to the wrapper placed around the scrollable.

Example 3

Adding items dynamically.

$('#scrollable3').scrollable({margin: 6, show_track: true, fade: false});

Make Scrollable.

You will need to refresh the Scrollable once the new item has been added. 

function addItem() {
    $('#scrollable3').append('<li>Item ' + ($('#scrollable3').children().length + 1) + '</li>');
    $('#scrollable3').scrollable('refresh');
}
